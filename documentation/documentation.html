Contact Section

In this section you can add your Google Map's latitude, longitude, map style, map marker image.

1. To add latitude and longitude, change the value of data-latitude and data-longitude attributes of the div with the id g-map respectively.
2. To change map styles, change the number of the data-mapstyle attribute of the div with the id g-map. Keep the value between 1-6, as 6 map styles are available.
3. To change map marker, change the url of the data-map-marker attribute of the div with the id g-map with your map marker image url.
4. To add your own email address for the contact form, change the value of the data-email-address attribute of the form with the id c-form with your own email address.

Before adding latitude and longitude make sure to add your Google Map API Key. You'll find this section in the index.html.
